About Us

Project Title: Smart Car Service & Garage Management System

Project Description:
This project is a full-stack web application designed to manage car service centers digitally. The system allows customers to book appointments, track service progress, and view invoices online. It also provides an administrative dashboard for managing services, mechanics, customers, payments, and revenue reports.

The system includes three main roles:
• Customer
• Mechanic
• Administrator

Main features include appointment booking, service management, invoice generation, reporting dashboard, customer reviews, and service tracking system.

The project aims to improve efficiency, reduce manual paperwork, and provide a modern digital solution for garage management.
